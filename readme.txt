内链tage标签，批量处理网站关键词及内链，可根据网站情况全部指定关键词超链到官网，或超链到分类（tag分类）页。根据SEO的不同进行设置，另外支持xlsx,xls两种表格格式批量上传关键词进行关联。
== Description ==

本插件适主要用于wordpress的网站文章关键词加超链，内链。

1、输入需要匹配所有文章的关键词，添加后，该关键词会将所有wordpress的网站文章关键词加超链，内链。（如果设置URL，则所有文章下的该关键词添加超链（内链）到指定地址。）

2、也拥有表格上传的批量功能，也可普通添加。

3、关键词超链（内链）有2种，第1种就是不设置URL，那么匹配的关键词自动会添加到TAG分类链接。第2种，如果设置指定ULR链接，则匹配的所有关键词添加超链指定链接。

4、内链关键词，可以加粗、文字变色等提升鲜明。

5、该功能主要是针对SEO内链需求较大的用户，需求小的建议购买SEO合集，可以登录www.seohnzz.com查看。

6、该插件放心替换，如果批量替换错误可以关闭或删除已添加的关键词。

== Installation ==

方式1：在线安装(推荐)
1. 进入WordPress仪表盘，点击“插件-安装插件”，关键词搜索“内链tag标签”，找搜索结果中找到“内链tag标签”插件，点击“现在安装”；
2. 安装完毕后，启用 `内链tag标签` 插件.
3. 通过“设置”->“内链tag标签” 进入插件设置界面.
4. 至此，该插件安装完毕，即可访问内链tag标签。

方式2：上传安装

FTP上传安装
1. 解压插件压缩包tagmanage.zip，将解压获得文件夹上传至wordpress安装目录下的 `/wp-content/plugins/`目录.
2. 访问WordPress仪表盘，进入“插件”-“已安装插件”，在插件列表中找到“内链tag标签”，点击“启用”.
3. 通过“设置”->“内链tag标签” 进入插件设置界面.
4. 至此，该插件安装完毕，次日即可访问百度站长后台查看内链tag标签。

仪表盘上传安装
1. 进入WordPress仪表盘，点击“插件-安装插件”；
2. 点击界面左上方的“上传按钮”，选择本地提前下载好的插件压缩包tagmanage.zip，点击“现在安装”；
3. 安装完毕后，启用 `内链tag标签` 插件；
4. 通过“设置”->“内链tag标签” 进入插件设置界面.
5. 至此，该插件安装完毕，即可访问百度站长后台查看内链tag标签。


== Frequently Asked Questions ==
=怎么使用内链tag标签？=
首先安装好插件以后即可在设置兰查看内链tag标签，添加需要的关键词即可。

=内链tag标签可以批量添加？=
肯定的，作为独立的插件就是为了可以批量处理大数据而开发。

=内链tag标签URL可以指定嘛？=
必须的，如果不设置，则是自动读取生成的tag分类链接。如果设置指定，则关键词批量超链到指定URL

=内链tag标签我不小心设置错误可以还原吗？=
是的，如果批量设置错误，可以删除设置错误的内链tag标签，已经匹配的则会失效而且跟以前一样。
